将纯文本转化为静态网站和博客

  • 简单

    无需数据库、评论功能，不需要不断更新版本--只用关心你的博客内容。

    Jekyll 是如何工作的 →

  • 静态

    只用Markdown(后Textile)、Liquid、HTML&CSS就可以构建可部署的静态网站

    Jekyll 模板指南 →

  • 博客形态

    自定义地址、分类、页面、博客内容以及自定义布局设计都是系统中的一等公民。

    迁移你的博客 →

只需要几秒钟就能让网站跑起来。

就这几个指令

~ $ gem install jekyll

~ $ jekyll new my-awesome-site

~ $ cd my-awesome-site

~/my-awesome-site $ jekyll serve

# => Now Browse to http://localhost:4000
